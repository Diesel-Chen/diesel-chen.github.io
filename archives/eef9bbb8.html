暖通二 地暖攻略

发表于 2025-04-16 分类于 装修 阅读次数： Valine：
本文字数： 5.3k 阅读时长 ≈ 5 分钟

以下是对水地暖（湿式地暖）系统的系统性总结，涵盖其工作原理、组成部分、选材建议、施工维护及系统设计要点，旨在帮助业主了解并选择适合的采暖方案。内容基于B站UP主@诡计从不拖更的视频内容整理，适用于家庭采暖，尤其在南方自采暖地区。

水地暖知识与选商总结

一、引言与背景

1.1 南北采暖差异

  • 历史背景：1950年代，受限于能源和运输条件，中国参照苏联经验，以秦岭—淮河线为界划分集中供暖区域（室外温度≤5℃，冬季天数>90天的北方地区）。
  • 现状
    • 北方：集中供暖（暖气片或地暖），室内温暖。
    • 南方：无集中供暖，需自采暖，常见方式包括空调、取暖器、暖气片、电地暖和水地暖。
  • 南方集中供暖难点：缺乏基础设施（热源厂、管网、换热站），建筑无保温标准，经济效益和环保要求限制。

二、南方常见自采暖方式对比

  1. 空调
    • 优点：制热快、效率高。
    • 缺点：热量不均、吹风干燥、头热脚凉、舒适度低。
  2. 取暖器（小太阳、暖风机、油汀等）
    • 优点：价格便宜、移动灵活、可局部加热。
    • 缺点：能耗高（相对空调），适合小范围或辅助。
  3. 暖气片（墙暖）
    • 原理：对流散热，高温供暖。
    • 优点：升温快（约半小时）、可随时开关、可明装（适合旧房改造）、可烤袜子。
    • 缺点：占空间、不美观、热量不均、舒适度低。
  4. 电地暖
    • 原理：发热电缆/电热膜将电能转为热能，地面辐射散热。
    • 优点：升温快、可随时开关、占用层高少（5-7cm）、安装维护相对简单便宜、舒适度高（较暖气片）。
    • 缺点：能耗费用高，不适合大户型（建议50㎡以下考虑）。
  5. 水地暖
    • 原理：以水为热媒，低温供暖（供水约50℃，回水约40℃），地面辐射散热。
    • 优点：供热均匀、舒适度最高（公认，温足凉顶）、不占墙面空间、美观、24小时常开情况下能耗费用最低。
    • 缺点：升温慢、占用层高较多（常规8-10cm，干式较少）、设备安装费用较高、不适合频繁开关。

选择逻辑总结

  • 精装/老房改造：暖气片。
  • 毛坯/全屋翻新：优先地暖。
  • **小户型 (<50㎡)**：可考虑电地暖。
  • **大户型 (>50㎡)**：优先水地暖。
  • 频繁开关：暖气片/电地暖。
  • 24小时常开：水地暖。

市场趋势

水地暖因其舒适度、能源利用（配合清洁能源热源）、适应中大户型等优势，已成为北方集中供暖和南方自采暖市场的主流选择。后续内容聚焦水地暖。

三、水地暖系统详解

工作原理简述

flowchart TD
    A[热源加热水] --> B[主管输送]
    B --> C[分集水器分水]
    C --> D[盘管回路辐射散热]
    D --> E[房间升温]
    E --> F[水经盘管]
    F --> G[分集水器集水]
    G --> H[主管回流]
    H --> A[热源循环]

    A --> I[保温层]
    I --> J[减少热量损失]
    E --> K[温控器]
    K --> L[执行器]
    L --> M[控制回路阀门开关]
    M --> N[实现温度调控]

热源加热水 → 主管输送 → 分集水器（分水） → 盘管回路辐射散热 → 房间升温 → 水经盘管 → 分集水器（集水） → 主管回流 → 热源（循环）。保温层防止热量损失。温控器联动执行器控制对应回路阀门开关，实现温度调控。

四大子系统

1. 热源系统 (Heat Source System)

主要类型

  • 市政集中供暖
  • 热泵（空气源、地源等）
  • 锅炉

核心设备：燃气壁挂炉（采暖炉）

功能：可同时提供采暖和生活热水（两用炉）。

类型选择

常规炉 vs. 冷凝炉

  • 常规炉：
    • 热效率85%-93%（二级能效）。
  • 冷凝炉：
    • 热效率>100%（一级能效），回收烟气潜热，更节能，尤其在低负荷运行时优势明显。

全预混冷凝炉

  • 燃烧充分，效率最高（105%-109%）
  • 缺点：价格贵、对燃料/空气/水质有要求、需专业维护

半冷凝炉（二次冷凝）

  • 效率中等（99%-105%）
  • 技术类似常规炉+烟气回收
  • 价格介于两者之间

常见误区纠正

  1. 误区1：半冷凝与全预混效率差不多。
    • 反驳：低负荷下全预混优势明显。
  2. 误区2：暖气片高温不适合冷凝炉。
    • 反驳：现代暖气片可用较低水温，冷凝炉依然节能。
  3. 误区3：冷凝炉用十几年省回成本。
    • 反驳：国标规定燃气具8年报废，应基于此周期计算。

锅炉选择建议

面积/气价

  • 大户型（>36kW）只能选冷凝
  • 中户型看气价：
    • 气价高则>70㎡建议冷凝
    • 气价低如重庆<100㎡可考虑常规炉

预算

  • 预算充足直接上全预混，更环保安全先进

未来趋势

  • 全预混是趋势
  • 半冷凝为过渡产品
  • 考虑未来气价上涨，有远见可选全预混

品牌

  • 德系：威能、菲斯曼、博士等
  • 意系：阿里斯顿、法罗力等
  • 日系：林内、能率
  • 其他：史密斯、德地氏、八喜、博途
  • 国产：美的、万和、小沃等

建议：无绝对技术壁垒，国内外大牌皆可，关注经销商服务能力。

生活热水功能（两用炉）

优点

  • 性价比高（省一个热水器钱）

缺点（对比专用燃气热水器）

  1. 水量偏小（28kW炉≈14L热水器）
  2. 恒温精度稍差（±1℃ vs ±0.1℃）
  3. 二次启停冷热水问题处理不如专用热水器

建议：

  • 对大多数用户OK
  • 高要求用户（多卫、浴缸、大花洒）可考虑独立高端燃气热水器或空气源热泵+燃气热水器

2. 控制系统

2.1 分集水器

功能：

  • 分水器（供水，红色）分配热水至各回路
  • 集水器（回水，蓝色/白色）回收热水

部件：

  • 支管（50mm间距）
  • 阀门（分水器带流量计，集水器带电热执行器）
  • 主阀门、排气阀、泄水阀、过滤器、压力表、温度计

选购要点（参考《GB/T 29730-2013》《GB/T 5231-2022》）

材质
  • 黄铜：
    • 最常见，壁厚4mm
    • 一体成型，避免漏水
  • 不锈钢：
    • 适合大管径（DN40）
    • 耐腐蚀
  • 高分子塑料：
    • 高端，耐腐蚀
    • 可供暖供冷

建议：优先大品牌，黄铜性价比高，未来不锈钢/塑料或成主流。

阀芯
  • 高位阀芯（防堵塞）
流量计
  • 建议带流量计，便于调试和调节各回路流量
品牌推荐

瑞好、菲索、沃茨、曼瑞德、欧文托普、意瓦、卡莱菲、嘉科米尼、玻玛、丹佛斯、乔治费歇尔、日丰、伟星

2.2 温控器与电热执行器

功能：

  • 温控器检测房间温度
  • 控制电热执行器开关集水器阀门
  • 实现分室控温

分室控温：每房间一个温控器，单独控制对应回路

2.3 联动控制器

功能：

  • 当所有回路关闭时，联动控制器关闭采暖炉
  • 避免干烧和频繁启停

现状：多数家装未配联动控制器，需业主主动要求

建议：分室控温必配联动控制器

2.4 智能家居接入

方法：

  1. 更换智能温控面板
  2. 支持协议后连接地暖设备（温控器、联动控制器）

注意：确保智能面板连接电热执行器和联动控制器

建议：无经验者可找智能家居商家对接暖通商

3. 管道系统 (Piping System)

分类

  • 主管：采暖炉至分集水器（供水管+回水管）
  • 盘管：分集水器支管至地面散热回路（每回路覆盖10~15㎡）

材质辨析

PPR

  • 优点：便宜、耐热
  • 缺点：硬、接头多（热熔依赖手艺）、非阻氧
  • 总结：基本不用于地暖盘管和主管（特殊情况：大管径地源热泵主管可能用）适合天氟地水大管径主管（DN40）

PERT（耐热聚乙烯）

  • 柔韧性好、耐热
  • 可热熔可卡套连接
  • 主管盘管均可用
    • I型 70℃（常用，性价比高）
    • II型 95℃（性能更强，工业用多）

PEX（交联聚乙烯）

  • 性能优异，耐压耐热
  • 卡套连接
    • PEX-a：性能足够家装，韧性好，施工方便。推荐PEX-a（大品牌）
    • PEX-b：环保性相对差，应用较少
    • PEX-c：物理交联，干净，性能好，但管材硬，施工难度大

阻氧管 (Oxygen Barrier Pipe)

作用：增加EVOH阻氧层，防止氧气渗入系统，减缓金属部件腐蚀。建议选用。

  • 三层：内管+粘接剂+阻氧层，表面阻氧层易划伤
  • 五层：内管+粘接剂+阻氧层+粘接剂+外管，阻氧层保护更好

必要性：有必要，虽系统内有少量氧，但持续渗入会产生问题。但无需过度焦虑，地埋环境氧含量低，且系统会换水

铝塑管 (PEX-Al-PEX等)

优点：

  • 中间有铝层，完全绝氧
  • 耐热耐压性好

缺点：

  • 散热性变差（不适合盘管）
  • 管材硬、弯曲易受损

适用：更适合用作主管或明装暖气片管道，不适合地暖盘管

管道选择总结与建议

必要性：阻氧管减少氧气渗入，保护金属件免腐蚀

推荐：

  • 盘管用PERT I型（3层阻氧）
  • 主管用PEX-a（5层阻氧或铝塑管）

核心观点：

  • 选对品牌比纠结具体型号更重要
  • 正规大品牌地暖管材非常可靠（设计寿命50年，质保长），极少因质量问题漏水
  • 避免小作坊产品

品牌推荐

瑞好、乔治费歇尔、伟星、金牛、金德、日泰、日丰、菲时特、波玛、沃茨、欧博诺、嘉科米尼、美尔固、保利、九牧

管道寿命

标准：

  • 《ISO10508》《GB/T 18991-2003》规定管道寿命50年（40℃×20年+60℃×25年+70℃×1年+20℃×余年+100℃×100小时）

质保：大品牌管道质保50年，品质可靠

4. 保温系统 (Insulation System)

作用：保温隔热，使热量向上辐射，均匀散热

4.1 传统湿式地暖辅材（4+2结构）

  1. 保温板：
    • 铺在最底层
    • 常用EPS（苯板）或XPS（挤塑板）
    • XPS性能更优
    • 推荐绿羽”白晶板”（高性能EPS）
  2. 铝箔胶带：粘贴保温板缝隙
  3. 反射膜：
    • 铺在保温板上，管道下
    • 向上反射热量
    • 建议用纯铝镜面
  4. 边界保温条：
    • 贴墙边
    • 侧向保温，提供伸缩空间
    • 用EVA材质
  5. 卡钉：固定地暖管
  6. 硅晶网（玻纤网）：
    • 铺在管道上，回填层内
    • 增强回填层强度、防开裂、助散热

4.2 模块地暖

特点：将保温、反射、卡管结构集成一体，施工便捷标准化

  • 蘑菇头式（蘑菇板）：
    • 管道卡在凸起结构间
    • 通常是湿式（需回填）
  • 预制沟槽式：
    • 管道嵌入预开好的槽内
    • 通常是干式（免回填）

4.3 湿式地暖 vs 干式地暖

湿式地暖（传统+蘑菇头类）

优点：

  • 成熟
  • 价格低（100~200元/㎡）

缺点：

  • 占层高8~10cm
  • 施工复杂

干式地暖（预制沟槽类）

优点：

  • 省层高2cm（6~8cm）
  • 施工简单
  • 升温快

缺点：

  • 价格高
  • 需提前找平
  • 无回填层蓄热差

适用场景

  • 湿式：
    • 层高宽裕
    • 铺瓷砖
  • 干式：
    • 层高受限
    • 铺木地板
    • 天氟地水

4.4 保温系统选择建议

  1. 传统湿式：
    • 技术成熟、性能稳定、价格便宜
    • 国内家装主流，性价比高
  2. 模块湿式（蘑菇板）：
    • 施工简单，品质均一
    • 价格稍高
  3. 模块干式（预制沟槽）：
    • 最贵，层高最薄，升温最快
    • 适用场景：
      • 对层高敏感
      • 铺木地板
      • 不介意升温快/蓄热差的情况
    • 不适合：
      • 层高充裕
      • 铺瓷砖的情况

四、地暖施工与维护

施工流程（以传统湿式为例）

参考《12K404》《12N1》《DB13T/1825-2013》等标准：

准备

  1. 分集水器安装：
    • 居中隐藏（墙体/橱柜）
    • 确保回路长度均衡
  2. 地面要求：
    • 干燥、无杂物
    • 平整度误差≤5mm（干式≤2mm）

铺设

  1. 贴边界保温条
  2. 铺保温板（2cm），用铝箔胶带粘接
  3. 铺反射膜
  4. 盘管（DN16间距150mm，DN20间距175mm），用卡钉固定

盘管要求：

  • 离墙150~300mm
  • 无接头
  • 单回路≤100m（覆盖≤15㎡）

伸缩缝

  • 进深>6m或面积>30㎡时设置，与盘管方向一致
  • 门口设置，方向与门洞一致，交叉处加柔性套管

打压测试

  1. 第一次（铺管后）：0.8MPa，1小时压降≤0.05MPa
  2. 第二次（回填后）：同上
  3. 第三次（入住前）：同上

回填

  1. 材料：豆石混凝土（1水泥：2~2.5黄沙：3豆石）
  2. 厚度：4cm，与边界保温条齐平
  3. 养护：5~7天，洒水固化
  4. 装饰面层：铺瓷砖（2cm）或木地板（2~4cm）

后期维护

  1. 锅炉保养：
    • 冷凝炉1-2年/次
    • 常规炉3-4年/次（理论应同冷凝炉）
  2. 管道清洗/换水：3-4年/次
  3. 电热执行器更换：约7-10年

五、特殊问题探讨

地面材料：瓷砖 vs 木地板

瓷砖

  • 优点：
    • 无甲醛
    • 导热好
    • 耐用
  • 优选

木地板

可用，但需注意：

  1. 环保：
    • 避免强化复合地板
    • 选环保等级高（E0/ENF）的实木复合或纯实木地板
  2. 导热：
    • 选薄的（实木复合15mm优于纯实木18mm）
    • 用薄的带孔地暖专用防潮垫
  3. 稳定性：
    • 选锁扣式
    • 铺设时留足伸缩缝（实木复合1cm，纯实木2-2.5cm）

综合看：实木复合地板比纯实木更适合地暖

地暖专用标识：查看是否符合国标GBT 35913（实木）或GBT 41547（木质）。若无，则可能只是营销术语

卫生间装地暖

可行性：可以装，但工艺复杂

要求：

  1. 需做两次防水（地暖前、回填后）
  2. 门口做止水坎

难点：

  • 防水和地暖属不同工种
  • 责任划分难
  • 商家通常不建议

替代方案：只装干区，配合风暖浴霸、电热毛巾架

六、行业现状批判与选商建议

行业问题（以罗兰朵案例）

普遍现象：

  • 经销商重产品销售、轻系统设计
  • 重宣传、轻技术服务

后果：

  • 缺乏系统思维
  • 即使好产品也可能因方案设计不合理（如锅炉选型错误、未做二次系统、缺少必要组件等）导致效果差、能耗高、故障频发
  • 最终损害用户体验和行业声誉

人员素质：很多从业人员缺乏专业知识，工程师配备不足

选择靠谱经销商（暖通服务商）的建议

  1. 专业团队：
    • 必须有懂技术的工程师负责方案设计并能直接沟通
    • 而非仅有销售
  2. 辅材透明：
    • 要求提供详细的辅材配置表（品牌、型号）
    • 了解其用料标准和定位，便于对比
  3. 施工标准：
    • 询问遵循的具体施工标准（国标/行标/企标）
    • 是否严格执行
    • 避免”凭经验”施工
  4. 公司实力：有充足备货通常意味着业务量大、实力较强
  5. 自身学习：
    • 掌握本总结中的知识点
    • 能有效识别忽悠
    • 主动提出需求（如联动控制器）

七、结语

地暖本身是成熟可靠的技术，能极大提升冬季舒适度。问题的关键在于找到专业、负责的服务商，进行合理的系统设计和规范的施工安装。业主自身也应加强了解，才能做出明智的选择，避开陷阱。
