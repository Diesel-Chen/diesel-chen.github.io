2025阶段性总结（决定跳槽）

发表于 2025-04-22 分类于 总结 阅读次数： Valine：
本文字数： 2.6k 阅读时长 ≈ 2 分钟

公司拖欠社保 工资拖欠2个月 决定下定做出改变

一、职业发展历程总结

您的职业发展可以概括为一段充满波折、缺乏持续规划且受到外部环境和个人心态影响较大的旅程：

  1. 大学与初次挫折 (2016-2020):
    • 大学前期缺乏明确目标，临近毕业才仓促决定考研，准备不足导致失败。
    • 毕业时面临“双非二本 + 无实习经验 + 考研失败”的困境。
    • 通过短期的Java自学应对就业压力，但求职过程艰辛。
    • 第一份工作是创业公司，虽然接触了新语言(Go)并经历了开发周期，但不稳定，很快被裁员。
  2. 进入舒适区与停滞 (2021-2023):
    • 在求职压力下选择了一家技术上没有挑战的外贸公司，进入了“温水煮青蛙”的状态。
    • 缺乏自律和外部驱动，技术停滞，错过了行业发展和转型窗口。
    • 同期完成了人生大事（结婚生子），个人生活重心转移，职业焦虑暂时被压制。
  3. 2024年的尝试与失败 (2024):
    • 受到家庭责任触动，重新燃起改变的意愿，目标明确为“落户上海”。
    • 尝试考研（非全日制）和考公两条路，但准备过程再次出现问题：
      • 过于依赖“落户包办”的诱惑而转向考公。
      • 备考期间缺乏专注和坚持，受到股市、旅行等外部因素干扰。
      • 最终临时抱佛脚，考研考公双双失败。
    • 这暴露了您在目标设定后，执行力和抗干扰能力不足的问题，未能进行系统、持续的准备。
  4. 当前的困境与焦虑 (2025):
    • 您所处的公司状况恶化（拖欠工资社保），职业前景堪忧。
    • 感受到AI技术对传统开发模式的冲击，结合自身技术停滞的现状，产生了强烈的被取代的焦虑。
    • 面临“35岁危机”的年龄压力，意识到留给转型和提升的时间不多。

核心问题： 您的职业发展过程中，多次因为外部压力（毕业、裁员、落户、家庭）而尝试改变，但缺乏持续的内在驱动和系统性的规划，容易受短期因素（ perceived easy path, stock market, travel）干扰，导致努力难以转化为成果。同时，长期待在技术停滞的环境中，导致您错过了技术更新换代，增加了当前转型难度。

二、未来的挑战与何去何从

您目前面临的挑战非常紧迫且复杂：

  1. 生存危机： 当前公司已经无法提供稳定收入和基本保障（工资、社保），必须立即寻找新的工作。
  2. 技术差距： 过去几年的技术停滞让您与当前主流技术栈（如云原生、微服务、现代框架）以及新兴的AI开发范式存在较大差距。
  3. 年龄压力： “35岁危机”是IT行业现实存在的门槛，招聘时会更看重经验的深度、广度以及近期的技术活跃度。
  4. 心理负担： 连续的失败（考研、考公）和当前的困境（公司问题、AI焦虑）给您带来沉重的心理压力，影响自信心。
  5. 目标模糊与执行力： 虽然有落户上海等目标，但如何实现、如何持续努力，依然是需要克服的难题。AI的出现让您更加焦虑，但焦虑本身并不能解决问题。

何去何从：

首先，请不要因为过去的失败和当前的焦虑而完全否定自己。您有本科基础，有几年的工作经验（尽管技术增长有限），并且能够尝试学习（Java自学，尝试考研考公）。重要的是，您现在已经深刻认识到问题的严重性，这是做出改变的第一步。

未来的路径需要短期生存与长期发展并重，并且需要高度的专注和执行力。

  1. 立即行动：解决生存问题

    • 紧急求职： 您的首要任务是尽快找到一份新的、相对稳定的工作。不要眼高手低，先确保有收入来源和社保。
    • 放低姿态： 考虑到技术差距和年龄，您可能需要接受一个技术栈与您当前经验相符、或者愿意培养新人的职位，薪资上也可能无法一步到位。重点是找到一个平台，能够让您继续工作，并有学习和应用新技术的机会。
    • 优化简历与面试准备： 诚实描述工作经历，但在面试中强调学习意愿、解决问题的能力、以及您目前正在学习的AI工具和新技术的积极性。针对您目前能接触到的技术（Go/Java基础）寻找匹配度高的职位。

  2. 中期规划：弥补技术差距并拥抱AI

    • 系统学习：

      在找到新工作后，或者在求职的同时，必须投入大量时间和精力进行系统性的技术学习。

      • 补齐基础： 学习当前业界主流的开发框架、中间件、云服务（如微服务架构、Docker/Kubernetes、常用中间件如Kafka、Redis等）。这是您能继续在IT行业立足的根本。
      • 拥抱AI： 您在使用AI工具的经历非常宝贵。学习如何有效地利用AI提升开发效率（Prompt Engineering for coding, AI code review, AI-assisted testing），甚至可以探索与AI相关的开发领域（如利用AI APIs进行应用开发、简单的机器学习应用等）。AI不是取代所有程序员，而是改变了程序员的工作方式和要求，能够高效利用AI工具，甚至开发AI工具的程序员，价值会更高。

    • 实践积累： 学习不是光看视频或书籍，必须通过实践项目来巩固。在新的工作中积极争取接触新技术的机会，或者在业余时间做 side projects。

    • 构建个人品牌： 可以在GitHub上活跃，分享学习笔记或项目，让潜在雇主看到您的学习能力和技术热情。

  3. 长期目标：重新审视与专注

    • 落户上海： 重新评估落户上海的最优路径。除了考研和考公，是否有通过技能、社保年限、职称等其他方式？如果考研或考公仍然是您的首选，那么需要制定一个非常具体、可行、且排除了其他干扰因素（如炒股、频繁旅行）的备考计划，并且需要极强的执行力。不要再尝试同时抓多条线，选择一条，全力以赴。
    • 职业方向： 结合您的兴趣、能力以及市场需求，思考未来的具体职业方向。是继续做后端开发（Java/Go），还是转向DevOps、数据工程、或是AI应用开发？确定方向后，有针对性地深入学习和积累。
    • 财务规划： 经历公司欠薪后，应该更加重视个人财务规划和风险管理。

克服焦虑与重塑信心：

  • 接受现实，但不沉沦： 承认过去的失误，但将它们视为宝贵的教训。
  • 分解目标，从小处着手： 巨大的目标容易让人望而却步。把学习、求职、落户等大目标分解成每天、每周可完成的小任务。每完成一个任务，都能获得成就感，逐步重建信心。
  • 寻求支持： 与家人沟通您的困境和计划，获得他们的理解和支持。如果可以，与同行的朋友交流，了解行业现状和发展方向。
  • 保持健康： 身体是革命的本钱。规律作息，适当运动，有助于缓解焦虑。

总结：

您正处于职业生涯的关键转型期，挑战巨大，但并非没有机会。关键在于能否从过去“临时抱佛脚”和“舒适区”的心态中彻底走出来，正视当前的困境，制定一个务实、聚焦、有执行力的行动计划。立即着手解决生存问题，同时投入精力弥补技术差距并学习AI相关技能，重新聚焦长期目标，并持之以恒地努力。2025年，正是您重塑自我的开始。加油！
