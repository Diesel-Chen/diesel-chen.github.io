窗户选购

门窗选购全攻略

窗型选择

开启方式

  • 推拉窗：通风好但密封差，不推荐主流使用
  • 悬窗：工装为主，家装可搭配平开窗五金
  • 平开窗（主流选择）：
    • 内开窗：北方主流，推荐加装内开内倒五金
    • 外开窗：南方常见，需配防坠器

计更推荐内开窗的三大理由：

  1. 空间问题可通过设计解决（180°开启/内倒模式）
  2. 纱窗外装更美观实用（避免室内冲突）
  3. 性价比更高（外开窗需额外转换框）

材质类型

材质 特点 适用场景
木窗 高端情怀款，需选大品牌 预算充足的实木爱好者
断桥钢 超高端（7k+/㎡），寿命极长 豪宅传承需求
塑钢窗 保温性能最佳，新国标已解决质量问题 节能首选（但目前定价偏高）
断桥铝 市场绝对主流 90%家庭的选择

性能指标（五性）

  1. 基础性能：

    • 抗风压（安全）
    • 水密性（防水）
    • 气密性（防风）

  2. 改善性能：

    • 保温性（K值，建议<2.5）
    • 隔声性（Rw+Ctr≥45dB）

注：检测报告获取困难，建议优先选择全系送检的一二线品牌

核心配置

型材配置

  • 关键参数：
    • 总宽度（75/80/90等系列）
    • 隔热条宽度（PA66GF25材质，24-35mm合理）

避坑指南：

  • 铝材材质/壁厚（新国标1.8mm）无需过度关注
  • 腔体填充/组角工艺对性能影响有限

玻璃配置

保温方案：

配置 效果 建议
中空层 双层基础，三玻降20%K值 北方必选三玻两腔
暖边条 4SG最优（氩气留存80%+） 改善型住房推荐
Low-E 单银/双银反射红外线 南方建议双银

隔音方案：

  • 基础：三玻两腔
  • 进阶：三玻夹胶一中空（万金油）
  • 顶级：四玻双夹胶一中空（计更同款）

五金系统

  • 优选隐藏合页（保障密封完整性）
  • 执手感观优先（无底座设计更美观）
  • 电动窗可接入智能家居系统

纱窗选择

  • 金刚网（0.7mm，防护强但挡光）
  • 高透网（0.3mm，推荐多数家庭使用）

商家选择指南

类型 价格区间 特点 适合人群
街边店 500-800元/㎡ 无品牌保障 预算极度紧张
本地小厂 800-1000元/㎡ 工艺不稳定 基础需求
全国品牌 1200+元/㎡ 注意经销商水平 品质追求者
B站带货 视配置而定 服务专业透明 高配需求用户

计更特别提示：线下门店普遍专业度不足，建议优先考虑B站专业UP主渠道

服务全流程

  1. 签约前：

    • 免费量尺设计（需与室内方案协调）
    • 报价方式：洞口面积+开启扇费用

  2. 合同要点：

    • 必写：可测量参数、五金品牌、安全条款
    • 勿写：无法鉴定的原材料承诺

  3. 安装验收：

    • 重点检查：型材/玻璃厚度、五金操作、密封完整性
    • 获取《安装验收表》（公众号领取）

  4. 售后服务：

    • 合理质保期：5年左右
    • 定期保养可延长使用寿命

窗户选购指南详细版本

1. 窗型

窗型分为开启方式和材质两大类，影响窗户的功能、外观和适用场景。

1.1 开启方式

推拉窗

  • 类型：左右推拉（常见）、上下推拉。
  • 优点：开启面积大，通风好，不占空间。
  • 缺点：密封性差，保温和隔音性能弱。新型提升推拉窗或侧压推拉窗密封性有所改善，但仍不及平开窗，且价格昂贵。
  • 推荐：不推荐，除非对性能要求极低（如面朝大海、春暖花开的房子）。

悬窗

  • 类型：上悬、中悬、下悬。
  • 特点：开启角度小，多用于工装（防盗或安全考虑）。家装中常作为平开窗的附加功能（内开内倒或外开上悬）。
  • 推荐：家装应用较少，视需求选择。

平开窗（市场主流）

  • 类型：内开窗（向室内开启）、外开窗（向室外开启）。
内开窗
  • 优点：
    • 擦窗方便，纱窗装在室外（美观、操作简便），可配内开内倒五金（通风且不占空间）。支持180°开启和防撞护角，减少磕碰风险。
  • 缺点：
    • 可能挡窗帘或室内物品（可通过设计规避）。
外开窗
  • 优点：
    • 不占室内空间，适合南方城市（政策宽松）。
  • 缺点：
    • 安全隐患（需配防坠器），纱窗装在室内（易挂猫爪、操作繁琐、美观性差），外开上悬五金成熟度较低，开启时需探身（对儿童不安全）。
纱窗对比
  • 内开窗：
    • 纱窗在室外侧，开关窗无需操作纱窗，美观且一致性好。
  • 外开窗：
    • 纱窗在室内侧，需先开纱窗再操作窗扇，影响美观（边框加宽、凸出），设计复杂（如口袋锁、滑盖锁）。
  • 价格：
    • 外开窗需额外转换框和型材，成本高于内开窗。
  • 政策：
    • 北方七层及以上建筑禁用外开窗，南方允许外开窗（需防坠器）。
  • 推荐：
    • 优先选内开窗（尤其南方用户），理由：
      • 开启方向：内开窗缺点可通过设计和五金（内开内倒、180°合页）解决，外开窗安全性和操作性较差。
      • 纱窗位置：内开窗纱窗室外侧，美观、方便；外开窗纱窗室内侧操作复杂且不美观。
      • 性价比：内开窗成本更低，性能相当。

1.2 材质

木窗

  • 特点：以指接板（松木、橡木等）为主，差异在漆和工艺。美观但需长期使用验证质量。
  • 适用：适合对实木有情怀的高预算用户，需选大品牌，不追求性价比。

钢窗

  • 特点：传统单层玻璃钢窗已淘汰，断桥钢窗为高端产品（国外品牌，手工加工，1万+元/㎡）。强度高、边框窄、寿命长。
  • 适用：豪宅业主，预算极高。

塑钢窗（PVC+钢衬）

  • 特点：保温隔热优于断桥铝，生产能耗低。过去因成本压缩（减少碳白粉、增加碳酸钙）质量差，市场印象较差。新国标（GB/T 28886-2023等）已提升品质，市场前景广阔。
  • 现状：优质塑钢窗价格高（1600+元/㎡），品牌少，市场接受度低。
  • 推荐：暂不推荐，待市场成熟。

铝木窗

  • 特点：性能与塑钢窗相当，价格高，适合追求木材质感的高预算用户。

断桥铝窗（市场主流）

  • 特点：铝合金+隔热条，平衡性能与成本，广泛应用于家装。
  • 推荐：本文后续内容均基于断桥铝门窗。

2. 性能

窗户性能类似“可开关的透明墙”，需兼顾美观、操作便捷、安全耐用、遮风挡雨、保温隔音。国标GB/T 8478-2020定义了七项性能，重点关注以下“五性”：

  • 抗风压性能（P3）：确保窗户在强风下不变形、不开裂。
  • 水密性能（ΔP）：防止雨水渗漏。
  • 气密性能（q1、q2）：减少空气渗透，防风防尘。
  • 保温性能（K值，W/㎡·K）：降低热量传递，K值越低越好（<2.5为保温型门窗）。
  • 空气声隔声性能（Rw+Ctr，dB）：隔绝噪音，数值越大越好（≥45dB为隔声型门窗）。

2.1 性能分类

  • 基础性能（抗风压、水密、气密）：与结构和安装相关，确保安全和基本防护。
  • 改善性能（保温、隔声）：与配置（如隔热条、玻璃）相关，需重点关注。

2.2 性能评估难点

  • 国标问题：均为推荐性标准（带“T”），厂商不强制执行。
  • 检测不足：
    • 厂商多用“Pro”“Max”等营销术语，不标注性能等级。
    • 小厂不做检测，大厂仅部分产品送检，检测报告可能不全面（缺保温、隔声数据）。
    • 增配后性能数据难以明确。
  • 解决办法：
    • 选大品牌：优先选择有五性检测报告的一二线品牌，口碑更可靠。
    • 了解配置影响：通过配置推算保温和隔声性能（详见配置章节）。

3. 配置

窗户由型材、玻璃、密封、五金和纱窗五部分组成，配置选择直接影响性能和价格。以下逐一分析。

3.1 型材

型材包括铝合金和隔热条，关注型材宽度和隔热条宽度。

型材宽度

  • 定义：指窗户截面总宽度（如75、80、90、120系列，单位mm）。
  • 作用：宽度越大，可容纳更厚玻璃、更多玻璃层或窗纱一体，适合高性能需求。宽度不直接影响性能，但与价格和产品定位相关（宽型材多为高端系列）。
  • 推荐：根据玻璃和纱窗需求选择，80mm以上适合窗纱一体或高性能配置。

隔热条宽度

  • 材质：聚酰胺66+25%玻璃纤维（PA66GF25），国标（GB/T 23615.1-2017）禁止用回收料。
  • 作用：隔热条断开铝合金热桥，降低K值（保温性能）。宽度越大，保温性能越好（边际效应递减，过宽牺牲强度）。
  • 常见规格：
    • 单体窗：24~35mm（70系列配24mm，80系列配34mm）。
    • 窗纱一体：因纱窗占用空间，隔热条较窄，保温性能略逊。
    • 市场最大：64mm（高端产品）。
  • 推荐：单体窗+外挂纱窗性价比更高，隔热条宽度根据型材匹配。

其他无需关注

  • 铝材材质：原生铝与再生铝无法鉴定，大品牌无需担心。
  • 铝材壁厚：新国标要求1.8mm（±0.1mm），一线品牌均达标。
  • 腔体设计、排水方式、喷涂技术、组角工艺、隔热条形状、腔体填充等：对性能影响小，无需纠结。

3.2 玻璃

玻璃占窗户面积大，对保温和隔声贡献显著。玻璃由原片厂（如南玻、台玻）生产，经深加工厂切割、钢化、合片后供门店采购。

保温配置（降低K值，减少热量传递）

  • 中空玻璃：
    • 定义：两片玻璃合片，中间形成空腔（如5+9A+5：两片5mm玻璃，9mm空腔）。
    • 类型：
      • 双层中空（标配）：南方勉强够用。
      • 三玻两腔（如6+12A+6+12A+6）：降低约20%K值，适合北方或高保温需求。
      • 四玻三腔：性能提升有限，仅限被动式建筑。
    • 空腔宽度：12mm比9mm降低2~3%K值。
  • 间隔边条：
    • 冷边条（铝条+分子筛+丁基胶+硅酮胶）：性价比高，工艺成熟。
    • 暖边条：
      • 硬性（如泰诺风聚丙烯+不锈钢）：保温性能稍优。
      • 柔性（热塑性TPSS，如4SG、TPE）：取代铝条等材料，降低人工要求，密封性强（4SG氩气25年留存80%+）。
    • 注意：4SG（科梅林授权）性能优于国产TPE，选购时确认材质，避免以TPE冒充4SG。
  • 充氩气：
    • 作用：惰性气体，降低热对流，K值降低5~8%。
    • 与TPSS关系：TPSS无分子筛，需充氩气以防Low-E氧化，密封性强确保氩气长效。
  • Low-E（低辐射玻璃）：
    • 作用：镀银反射红外热辐射，降低10~20%K值。
    • 类型：
      • 高透（透光率>70%）：家装主流。
      • 单银：北方标配。
      • 双银：南方推荐（三银无必要）。
    • 位置：
      • 南方：室外侧第二面（B面），反射室外热量。
      • 北方：室内侧第三面（C面），保留室内热量。
    • 推荐：标配Low-E，南方优先双银。

隔声配置

  • 三玻两腔：隔声性能较好，适合一般需求。
  • 夹胶玻璃：
    • 定义：两片玻璃间夹PVB胶片（干夹），单片PVB吸收约30dB噪音。
    • 推荐配置：
      • 三玻夹胶一中空：保温隔声兼顾，性价比高。
      • 四玻双夹胶一中空：隔声极致，适合噪音敏感用户。

外观配置

  • 普玻 vs 超白玻：超白玻透光率高，Low-E后与普玻效果接近。推荐：预算充足可作为最后升级。
  • 超大玻璃：
    • 特点：需更厚玻璃（10mm+），价格高，自爆率略高（约3‰，超白玻较低）。
    • 推荐：预算充足且追求外观，选北玻（低自爆率，承诺赔偿）。
  • 推荐配置：
    • 刚需：双层中空+冷边条+单银Low-E。
    • 改善型：三玻两腔+暖边条（4SG优先）+双银Low-E+充氩气。
    • 高要求：四玻双夹胶一中空+4SG+双银Low-E+充氩气。

3.3 密封

  • 材质：三元乙丙（EPDM）胶条。
  • 位置：
    • 玻璃与压线/窗扇间（室内外侧）：可用胶条或人工打胶，胶条更标准化。
    • 窗扇与窗框间：三道密封（室外止口胶条、等压胶条、室内止口胶条）。
  • 注意：
    • 三道密封足矣，多道密封为营销噱头。
    • 等压胶条与隔热条搭接≥3mm，确保垂直等温结构。
    • 明装合页破坏室内止口胶条，推荐隐藏合页。
  • 检验：检查胶条完整性，试纸抽拉阻力。

3.4 五金

  • 类型：执手、合页、传动配件、锁闭配件、提升块、防坠器、限位器等。
  • 特点：厂商多采购成熟五金（国产/进口），无需自研，质量可靠。
  • 关注点：
    • 合页：
      • 明装（柱式）：破坏密封，少用。
      • 隐藏（铰链）：支持180°开启，优先选。
    • 执手：无底座/小底座执手美观、手感佳。
    • 电动窗：支持智能家居（如米家），可实现自动开关场景。
  • 推荐：预算1000+元/㎡，国产五金已足够；2000+元/㎡，进口五金常见。

3.5 纱窗

  • 类型：
    • 卷轴式（隐形纱窗）：聚酯纤维，易损，无防护性。
    • 推拉纱窗：适配推拉窗，卡轨道。
    • 框中框：仅适配内开窗，需拆卸。
    • 平开纱窗（主流）：适配平开窗。
  • 纱网材质：
    • 金刚网：不锈钢+喷涂，丝径0.7mm，结实但遮光通风差。
    • 高透网：不锈钢，丝径0.3mm，28目，透光通风好，防护性稍弱。
  • 推荐：非高防护需求选高透网。

3.6 系统窗

  • 定义（GB/T 39529-2020）：采用系统化设计、满足性能要求的定型门窗。
  • 现状：定义宽泛，大小厂商均自称系统窗，概念意义减弱。
  • 意义：推动行业品质提升，选购时无需过分关注“系统窗”标签。

4. 商家

商家选择基于配置需求和预算，分为四类。

4.1 配置需求总结

  • 内开/外开：推荐内开窗，南方或北方低楼层可选外开。
  • 窗纱一体/单体+外挂：
    • 内开：单体+外挂纱窗性价比高。
    • 外开：窗纱一体（外挂纱窗不美观）。
  • 型材宽度：80mm+适合高性能或窗纱一体。
  • 玻璃：
    • 刚需：双层中空+冷边条。
    • 改善：三玻两腔+4SG+双银Low-E。
    • 高要求：四玻双夹胶一中空+4SG。
  • 五金：隐藏合页、无底座执手，考虑电动窗。
  • 纱窗：高透网为主。

4.2 商家类型

  • 街边店/小区店：
    • 价格：500~800元/㎡。
    • 特点：成本低，配置基础，服务有限，存在“见人下菜”。
    • 适合：低预算、对性能要求低，需专业知识砍价。
  • 建材市场/商场小厂门店：
    • 价格：800~1000元/㎡。
    • 特点：原材料达标，工艺（如TPSS）有限，服务一般。
    • 适合：中等预算，需关注细节。
  • 全国品牌门店：
    • 价格：1200+元/㎡。
    • 特点：品控可靠，但经销商水平参差不齐，报价差异大。
    • 适合：高预算，需多店对比。
  • B站带货：
    • 特点：与一线品牌合作，产品、服务、价格可靠，省心。
    • 适合：高配置需求，理性消费用户。

4.3 推荐

  • 高预算/高要求：优先B站带货或全国品牌门店。
  • 低预算：街边店，需深入学习配置细节。

5. 服务

服务分为签约前、签约付款、安装验收和售后四个阶段。

5.1 签约前

  • 量尺与设计：
    • 多免费，部分需订金（可退）。与室内设计师协作，明确外观需求，门窗商家完成细节设计。品牌门店可提供效果图/施工图及技术交底。
  • 报价：
    • 方式：洞口面积×单价+开启扇费用，或全包价。
    • 增配费用：超大玻璃、加厚玻璃、吊装费等。
    • 建议：充分沟通，确保设计满足需求。

5.2 签约付款

  • 合同：
    • 需写明：厚度、宽度、结构图、五金品牌、安全责任（与甲方无关）。
    • 无需写：玻璃原片品牌、原材料材质、性能（难鉴定）。
    • 性质：消费买卖合同（非业务承揽）。
  • 付款：
    • 分期（如5/3/2、5/4/1）或全款（视商家信誉）。
    • 建议：确认合同条款清晰，付款方式合理。

5.3 安装验收

  • 特点：验收专业性高，需业主自查（设计师、监理多不专业）。
  • 工具：参考《安装规范与验收表》（通过“诡计从不拖更”公众号获取）。
  • 建议：将验收标准写入合同，严格按规范检查。

5.4 售后

  • 现状：多由门店提供，厂商少承诺。
  • 年限：5年合理，终身售后多为噱头。
  • 保养：高端门店提供清洗、调五金、上油等服务，有助于延长寿命。
  • 建议：关注安装后短期问题，定期保养。
